Title Assistant Settings

This is added only when the complete title remains within eBay's 80-character limit. Leave it blank to disable the suffix.

Example

When enabled, the eBay shortcut searches the shop's existing description template for Artist, Title, Label and Catalogue Number, then replaces only their values. It does not recreate the template.

Ctrl+Shift+Y: Generate, save and copy on Discogs.

Ctrl+Shift+E: Fill the active eBay draft.

Ctrl+Shift+U: Open this settings page.
